## Overview

BioOPC is an AI-assisted career assessment tool designed for undergraduate students in research-oriented programs.

Instead of telling users whether they should pursue a PhD, BioOPC helps them better understand their research interests, motivation, resilience, and long-term career preferences through a structured assessment framework.

The project was inspired by real challenges faced by students in China's Fundamental Discipline Program (强基计划).

---

## Features

- 📊 Multi-dimensional career assessment
- 🧠 AI-assisted scoring framework
- 💬 Personalized feedback report
- 🎯 Decision support instead of decision making
- 🌐 Interactive Web Demo
